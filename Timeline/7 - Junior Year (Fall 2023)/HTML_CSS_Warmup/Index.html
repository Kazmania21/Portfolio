Web Development

In web development, the main tools for making a web page are HTML and CSS. If you want to know more about CSS, click here. In this section, we will discuss HTML.

HTML

HTML stands for HyperText Markup Language. It is the primary markup language used to create and format web pages. It is a powerful tool that serves as the foundation of the world wide web.

History

The idea for HTML was conceived in 1980 by CERN physicist Tim Berners-Lee. It was originally intented as a convenient way for CERN employees to share documents via hypertext. By 1995 HTML went from a CERN-specific technology to being maintained by the Internet Engineering Task Force. In 2000 HTML was made the international standard for webpages by the ISO and IEC committees. HTML 5 has been the lastest version in use from 2014 onward.

Example Components

  • The <p> element is used to make paragraphs

  • The <h1>-<h6> elements are used for headers

  • The <ul> element is used to make an unordered list (what you are currently looking at)

  • The <table> element is used to make tables, along with <tr> for the rows and <th>/<td> for the columns

  • The <div> element is used to semantically break up HTML code into sections
