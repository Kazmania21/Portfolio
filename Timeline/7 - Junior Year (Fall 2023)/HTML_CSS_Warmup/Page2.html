CSS

CSS stands for Cascading Style Sheets. It is used to take the elements created in an HTML file (or any XML file) and make them look more appealing.

History

The idea of CSS was proposed by H책kon Wium Lie in 1994. In 1996, The World Wide Web Consortium recommended CSS for creating style sheets due to its unique ability to combine multiple style sheets by cascading the styles.

Listed below are the different versions of CSS:

  • CSS1 - Allowed users to change the styles of fonts, colors, text spacing, element alignment, and box model assisted by the use of an element's id or class attribute.
  • CSS2 - Added positioning of elements, new font properties, and more.
  • CSS2.1 - Removed poorly supported features.
  • CSS3 - Extends on the capabilities with CSS2 using seperate modules or file to maintain backwards compatibility.
